#### OpenWrt_Build_x64项目的插件库

*  常用OpenWrt软件包源码合集，同步上游更新！

*  通用版luci适合18.06与19.07



#### 使用方式（三选一）：

1. 先cd进package目录，然后执行

```bash
 git clone https://github.com/gxnas/OpenWrt-Packages
```
2. 或者添加下面代码到feeds.conf.default文件

```bash
 src-git small8 https://github.com/gxnas/OpenWrt-Packages
```
3. lede/下运行 或者openwrt/下运行

```bash
git clone https://github.com/gxnas/OpenWrt-Packages package/openwrt-package
```
